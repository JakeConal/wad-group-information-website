About this project

This is a static HTML & CSS project created for a web apps assignment. It demonstrates a responsive layout, an image map that links faces to member pages, and accessible member information pages.

The team built the site with these goals in mind: clarity, accessibility, and a polished visual presentation suitable for class demonstration.
